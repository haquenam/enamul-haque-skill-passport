# Enamul Haque Skill Passport

Free assessment, recognition, and certificate verification portal for learners completing Enamul Haque’s open technology courses.

## Purpose

Enamul Haque Skill Passport is designed as the recognition layer across Enamul Haque’s open learning ecosystem. Learners may study through YouTube, GitBook, GitHub, practice resources, and open source learning tools. This portal will help them assess their understanding, validate course completion, and receive shareable recognition.

## Pilot Scope

The first pilot release focuses on the landing page and public positioning for the portal.

The first enabled assessment pathway will be:

* CSC For Non CSC Students
* Module 1 Completion Assessment
* 50 questions
* 60 minutes
* 70 percent pass mark
* Certificate with unique verification ID

## Future Recognition Pathways

Planned future pathways include:

* AI Integrator Course
* Cybersecurity 101
* Data Science Fundamentals
* Power BI For Beginners

## Recognition Positioning

This portal provides completion assessment recognition for open learning courses. It is intended for learning validation, progress recognition, and personal skill development. It is not a university degree, formal academic credit, or externally accredited professional qualification.
