Assessment Policy

Initial pilot rules

  • Question type: multiple choice questions.
  • Questions shown: 50.
  • Time limit: 60 minutes.
  • Pass mark: 70 percent.
  • Certificate: generated only after passing.
  • Certificate ID: unique verification ID planned for the certificate release.
Coverage Blueprint

Module 1 topics to prepare

The final question bank should be generated from these topic groups so the assessment remains balanced and fair.

Computer Science Basics

Course purpose, CS mindset, digital literacy, computer science fundamentals, and the role of computer scientists.

CS Vs Engineering

Difference between computer science, computer engineering, hardware focus, software focus, and problem solving.

History And Generations

Computer history, major generations, technology evolution, and why these changes matter.

Architecture And Hardware

CPU, input, output, processing units, memory, storage, and core computer architecture concepts.

Software And OS

System software, application software, operating systems, file management, GUI, CLI, and memory management.

Real World Digital Use

Computer types, real world uses, social media, search engines, SEO basics, and online awareness.
